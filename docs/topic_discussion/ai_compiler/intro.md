# AI Compiler 专题

## 专题概览
本专题用于沉淀贯穿 Part 1-4 的图优化、编译链路和自动优化策略。
AI Compiler 之所以重要，是因为当模型规模和系统复杂度上升后，很多性能问题已经不能只靠手写经验解决，而需要借助编译器把图级优化、算子融合、调度和代码生成组织起来。理解这个专题，才能更好地理解“为什么某些优化能落地，某些优化只能停留在概念里”。

## 专题内容
- Part 1：硬件与系统层的编译视野
- Part 2：PyTorch 图优化与执行语义
- Part 3：Triton / kernel 生成链路
- Part 4：CUDA / 系统层优化与落地
