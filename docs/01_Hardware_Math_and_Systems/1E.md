# Group 1E: Compiler Optimization and Hardware Ecosystem | 1E: 编译优化与硬件生态

## Group Overview | 组概览

这一组决定你后面会不会把“模型优化”只理解成 kernel 微调。真正的系统优化，往往要先看图、再看编译、最后才看硬件。阅读顺序和组内资产见下表，先从 [1D](./1D.md) 过来，再结合 [19](./19_Operator_Fusion_Introduction.md)。
