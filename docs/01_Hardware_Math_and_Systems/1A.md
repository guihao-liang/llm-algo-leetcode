# 1A: 数值基础与算力估算

本组解决“先要算什么”的问题，目标是建立显存、精度、参数量和 FLOPs 的底层直觉。

## 组概览

这一组先把“算什么、怎么算、怎么算得快不快”说清楚，是 Part 1 的数量级底座，也是不少后续章节的共同前提。阅读顺序和 Part 级前导路径见 [Part 1 导学](./intro.md)。

## 学习路径

### 推荐顺序

- 主线先看 `01 / 02`，把数据格式、精度、参数量和 FLOPs 的数量级先立住。
- 同类章节再看 `21 / 22`，把量化和 MoE 作为主线的变体继续补完。

### 面向后续部分

- 基础认知层：先把 `01 / 02` 看完，再进入 Part 2 / Part 3 的参数量、显存和算力推导。
- 变体补充层：`21 / 22` 主要补量化和 MoE 的判断直觉，方便后续做压缩、推理和训练侧分析。

- 具体运行条件以后续单页说明为准，不按组一刀切。
