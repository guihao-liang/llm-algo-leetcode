# Group 1A: Numerical Foundations and Scale Estimation | 1A: 数值基础与算力估算

本组解决“先要算什么”的问题，目标是建立显存、精度、参数量和 FLOPs 的底层直觉。

## Group Overview | 组概览

这一组先把“算什么、怎么算、怎么算得快不快”说清楚，是 Part 1 的数量级底座，也是不少后续章节的共同前提。阅读顺序和 Part 级前导路径见 [intro](./intro.md)。

## Learning Path | 学习路径

### Recommended Order | 推荐顺序

- 主线先看 [01](./01_Data_Types_and_Precision.md) 和 [02](./02_LLM_Params_and_FLOPs.md)，把数据格式、精度、参数量和 FLOPs 的数量级先立住。
- 同类章节再看 [21](./21_Quantization_Theory_and_INT4_INT8.md) 和 [22](./22_MoE_Parameter_and_Compute.md)，把量化和 MoE 作为主线的变体继续补完。

### Next Steps | 后续衔接

- 基础认知层：先把 [01](./01_Data_Types_and_Precision.md) 和 [02](./02_LLM_Params_and_FLOPs.md) 看完，再进入 Part 2 / Part 3 的参数量、显存和算力推导。
