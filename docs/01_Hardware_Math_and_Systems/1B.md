# Group 1B: Single-GPU Hardware and Memory Optimization | 1B: 单卡硬件与访存优化

本组解决“单张 GPU 怎么算得快”的问题，核心是理解 GPU 架构、内存层次和 Attention 访存优化。

## Group Overview | 组概览

这一组先把 GPU 架构、内存层次和 Attention 的访存路径说清楚，是 Part 1 里“看见算子慢在哪里”的起点，也是后续相关优化页面的共同前提。阅读顺序和 Part 级前导路径见 [intro](./intro.md)，从 [1A](./1A.md) 进入，再按需要继续到后续组页。
