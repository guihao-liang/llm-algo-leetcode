# Group 4.1: CUDA Programming Basics | 4.1: CUDA 编程基础

本组聚焦 CUDA C++ 基础、原生 kernel 和 shared memory，帮助读者建立 GPU 编程与执行模型直觉。

## Group Overview | 组概览

本组从最基础的 CUDA kernel 写起，逐步进入 shared memory、bank conflict、Tensor Core MMA 和 warp-level primitives，建立从程序到硬件执行的最小闭环。
