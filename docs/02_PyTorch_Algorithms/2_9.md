# Group 2.9: Project Practice | 2.9: 项目实战

本组用于承接综合项目、性能对比与训练分析，是 Part 2 的主要收束层和缓冲区。当前按“核心 3 个项目 + 扩展 3 个项目”组织，其中核心 3 个已经补出项目骨架，扩展 3 个仍先占位；核心项目优先覆盖训练、推理和训练性能分析，扩展项目优先覆盖 profiling 闭环、分布式基准和量化部署，后续按优先级逐个补齐。
