# 组队学习专题

## 专题概览
`team_study` 用于沉淀每一次组队学习的完整资料。一次组队学习对应一个独立文件夹，通常围绕某个 Part 或某组任务展开。

## 一次组队学习包含什么
- 学习计划
- 每个 Task 对应的优秀学习笔记
- 助教出题与题目整理
- 组队讨论、复盘与补充记录
